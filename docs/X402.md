# x402 Payment — Go-Live Guide

The endpoint `GET /api/launches/latest` is wired for x402 via `withX402` (settles
only on a successful response). It's **off by default** (open/free) so the value
layer runs with zero config. Here's how to turn it on and verify real payment.

## What's already done
- `src/app/api/launches/latest/route.ts` wraps the handler with `withX402` when
  `X402_ENABLED=true` and `X402_PAY_TO` is set.
- Verified: with x402 on, an unpaid call returns **HTTP 402** + a valid
  `accepts` payment-requirements body (scheme `exact`, USDC, our price + payTo,
  `discoverable: true` for the x402 Bazaar crawler).

## Step 1 — Test on Base Sepolia (free, no real money)
Testnet uses the default x402.org facilitator — no CDP keys needed.

1. Create two test wallets (e.g. `cast wallet new`, or any tool): a **seller**
   (receives) and a **buyer** (pays). Note their addresses + the buyer's private key.
2. Fund the **buyer** with Base-Sepolia USDC — Circle faucet: https://faucet.circle.com
   (the `exact` scheme is gasless for the payer via EIP-3009, so no ETH needed).
3. Start the server in paid testnet mode:
   ```bash
   X402_ENABLED=true X402_NETWORK=base-sepolia X402_PRICE=0.03 \
   X402_PAY_TO=<seller-address> npm run dev -- -p 3001
   ```
4. In another terminal, run the buyer simulation:
   ```bash
   BUYER_PRIVATE_KEY=<buyer-pk> npx tsx scripts/buyer-test.ts
   ```
   Expect: `HTTP 200` + the scored launches JSON. Confirm USDC moved to the seller
   on https://sepolia.basescan.org.

## Step 2 — Go live on Base mainnet
Mainnet settlement needs a Coinbase CDP facilitator (free to sign up).

1. Get CDP API keys: https://portal.cdp.coinbase.com → set `CDP_API_KEY_ID` /
   `CDP_API_KEY_SECRET`.
2. Install the CDP facilitator: `npm i @coinbase/x402`.
3. In `route.ts`, pass the facilitator to `withX402(handler, payTo, routeConfig, facilitator)`:
   ```ts
   import { facilitator } from "@coinbase/x402";
   // ...
   return withX402(handler, payTo, routeConfig, facilitator);
   ```
4. Set env on your host (Vercel): `X402_ENABLED=true`, `X402_NETWORK=base`,
   `X402_PAY_TO=<your mainnet wallet>`, `X402_PRICE=0.03`, plus the CDP keys.

## Step 3 — Get discovered
- List the endpoint on **x402 Bazaar** / **Agentic.Market** (category: data + trading).
  The 402 response already advertises `discoverable: true` and an output schema.
- Publish a thin **MCP server** wrapper so MCP-based agents can add it as a tool.

## Pricing knobs
- `X402_PRICE` (USD of USDC per call). Default `0.03`. Premium/real-time tiers can
  be separate routes with higher prices later.
